## Remove Similar Images

This script receives as an argument a folder path and removes similar-looking images.

## Dependencies:
* OpenCV 4.5.5.64
* imaging_interview script (has helper functions for preprocessing and comparison between images; omitted here)


## Using this Repository

* Clone this repository.
* Install dependencies.
* Open a terminal in the repository folder. Run the following:
```
  python main.py <path_to_folder>
  ```
